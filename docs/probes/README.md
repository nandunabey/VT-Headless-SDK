# Probe Scripts

Standalone diagnostic scripts used during SDK development.
All require SteamVR running.
Not part of the core SDK.

| Script | Purpose |
|--------|---------|
| probe_tracker_identity.py | Dump all tracker properties via OpenVR |
| probe_openxr.py | Test headless OpenXR session (SteamVR) |
| probe_ports.py | Check port availability |
| probe_ws.py | Test WebSocket connection |
| probe_tracker_pipe.py | Test tracker data pipeline |
